## 📖 About

Nex Loader is an external project built for Windows.

### Features

* 👁️ Visuals
* 🎯 Aimbot
* ⚙️ Miscellaneous Features
* 🚀 Fast Loader
* 🔄 Active Development

> **Note**
>
> The included kernel-mode driver was developed by me and is still a work in progress. Bug fixes and improvements are being made regularly.

---

## ⚠️ Warning

The bundled kernel driver is currently **unsigned**.

Because of this, it may not load on systems that enforce driver signature verification.

---

## 📂 Installation

1. Download the latest release.
2. Extract the files.
3. Launch the application.
4. If you encounter issues, follow the troubleshooting steps below.

---

## 🛠 Troubleshooting

If the loader does not work:

1. Open the **Fixer** folder.
2. Run **requirement.bat**.
3. Wait for all required components to install.
4. Run **fixer.bat**.
5. Restart the application.

---

## 💬 Support

If you've completed every troubleshooting step and are still experiencing issues, feel free to contact me on Discord.

When reporting a problem, include:

* Windows version
* Screenshot of the error
* Description of what happened
* Any console or log output (if available)
