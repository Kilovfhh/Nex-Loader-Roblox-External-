# Nex Loader

[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?logo=github)](https://github.com/Kilovfhh/Nex-Loader-Roblox-External)

## Overview

Nex Loader is an external Windows application.

---

## Installation

1. Download the latest release.
2. Extract all files to a folder.
3. Run `Nex.exe`.
4. Follow any on-screen instructions.

---

## Troubleshooting

If you experience any issues:

1. Open the **Fixer** folder.
2. Run `requirement.bat` and allow it to finish installing any required dependencies.
3. Run `fixer.bat`.
4. Restart the application.
5. If the issue persists, contact support.

---

## Support

If you've followed all of the troubleshooting steps and are still having problems, feel free to reach out on Discord.

When asking for help, please include:

- Windows version
- Error message (if any)
- Screenshot of the issue
- A description of what happens when launching the application

This will help resolve your issue faster.
