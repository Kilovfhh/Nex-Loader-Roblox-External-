How to run Nex.exe
---------------------

Made by: nexa 
-------------------

1. Make sure to have TPM disabled, Secure Boot disabled, Core isolation disabled, kernal signing disabled.
2. Restart your pc
3. Turn off Real-Time Protection 
4. Launch any roblox game; Then launch Nex.exe and wait for it to load...


------------------
If you encounter any issues please contact me on discord AFTER you done these steps...


1. Open the folder called Fixer
2. Run the requirement.bat file and wait for all the stuff to be installed
3. Run the fixer.bat 
4. Once done retry running Nex.exe
5. This should be all
